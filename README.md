# Digital-Farm-Management-System
The project describes a Digital Farm Inventory System (DFMS) designed to help farmers manage resources efficiently.
It allows users to add, update, and remove items, to generate inventory reports and financial summaries.
The system improves asset tracking, cost monitoring, and decision-making on farms.
However, it has limitations such as manual data entry, local operation, and limited real-world testing.

Overall, the system provides a simple foundation for improving farm management, decision-making, and moving toward smart agriculture solutions.
